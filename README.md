# 🔢 LeNet-5 MNIST Classifier

Projeto de **Visão Computacional** utilizando **PyTorch** para classificar dígitos manuscritos do dataset **MNIST**.

---

## 🧠 Arquitetura

Implementação da arquitetura **LeNet-5 (CNN)**:

- Input: 28x28 pixels
- 2 camadas convolucionais
- 2 camadas de pooling
- 3 camadas totalmente conectadas
- saída com 10 classes

---

## 📊 Resultado

Accuracy obtida:

**98.7%**

---

## 📚 Dataset

Dataset utilizado:

**MNIST**

- 60.000 imagens de treino
- 10.000 imagens de teste
- imagens de dígitos manuscritos
